# Particle Journey

An interactive canvas experiment exploring movement, particles, and Voronoi-inspired visuals.

## About the project

Particle Journey started as a playful coding experiment. I wanted to explore how simple rules, motion, and visual systems can create something that feels organic and unpredictable.

## Built with

- HTML
- CSS
- JavaScript
- Canvas

## How to experience it

Open the live project, move around, and let the particles do their thing.

## Controls

- Press `H` to show or hide the performance information
- Use the record button to capture a 15-second animation

## Behind the scenes

The project uses a full-screen HTML canvas, with the visual logic handled in `voronoi.js`.
